# 🐛 Mites Identification App

## 📌 Overview

The **Mites Identification App** is an educational and practical tool designed to support the identification and understanding of key **beneficial predatory mites** used in Integrated Pest Management (IPM).

It focuses on widely used biological control agents that help reduce pest populations in greenhouse and agricultural systems.

---

## 🧬 Included Species

This project currently supports identification and reference data for the following mite species:

- *Amblyseius californicus*
- *Amblyseius montdorensis*
- *Amblyseius swirskii*
- *Amblyseius cucumeris*

## 🕷️ Species Details

### *Amblyseius californicus*
A resilient predatory mite effective against spider mites, particularly in hot and dry climates. Commonly used in fruit and vegetable production systems.

### *Amblyseius montdorensis*
A generalist predator targeting thrips, whiteflies, and spider mites. Performs well in diverse greenhouse environments.

### *Amblyseius swirskii*
A highly effective biological control agent against thrips and whiteflies, widely used in protected cropping systems.

### *Amblyseius cucumeris*
Primarily targets thrips larvae and is often introduced preventatively in early crop stages.
